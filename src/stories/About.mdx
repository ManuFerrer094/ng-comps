# Acerca de este proyecto

Este repositorio expone una colección de componentes reutilizables con una capa de estilo propia sobre Angular Material.

**Puntos clave:**

- Diseño: `src/theme/tokens.css` contiene las variables de diseño (colores, tipografías, espaciados).
- Tema Material: `src/theme/material-theme.scss` contiene la configuración y mixins de Material.
- Componente base: `src/app/components/button/mf-button.component.ts` es el `MfButton` listo para producción.

Puedes inspeccionar y reutilizar los archivos mencionados en el repositorio.
