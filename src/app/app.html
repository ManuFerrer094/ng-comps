This page explains how to install and start using ng-comps in any Angular project.

1) Install package

npm i ng-comps

2) Load base styles and design tokens

Add these imports in your global styles file (for example src/styles.css):

@import 'ng-comps/theme/tokens.css';
@import 'ng-comps/styles.css';

3) Use components in standalone Angular pages

Import only what you need for better tree-shaking.

import { Component } from '@angular/core';
import { MfButtonComponent, MfInputComponent } from 'ng-comps';

@Component({
  selector: 'app-example',
  imports: [MfButtonComponent, MfInputComponent],
  template: `
    <mf-input label="Email" placeholder="you@company.com" />
    <mf-button label="Save" variant="filled" />
  `,
})
export class ExampleComponent {}

4) Notes for production projects

  • The package is published with sideEffects: false for max tree-shaking.
  • Angular and RxJS are peer dependencies to avoid duplicated framework installs.
  • Import components directly from ng-comps; avoid wildcard barrel files in your app.
